# Assembly Drawing Requirements

This is a compatibility entrypoint for older references.

For new work, read `assembly/index.md`, then select the applicable module under
`assembly/standards/`. When no project, company, customer, national, or
industry standard is specified, use
`assembly/standards/generic-mechanical.md`.

Do not keep new standard-specific rules in this file. Add a focused module
under `assembly/standards/` and register it in `assembly/index.md`.
