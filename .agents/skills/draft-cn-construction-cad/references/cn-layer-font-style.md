# 中文图层、字体与样式

## 图层命名

默认使用中文图层名。推荐格式为 `专业-对象-状态`，状态可省略。示例：`建筑-墙体`、`结构-梁`、`给排水-排水-新建`、`建筑-墙体-拆除`。

若项目已有图层标准，严格沿用项目标准；否则使用下表作为起点。

## 字体选择

1. 先调用 `get_text_styles()`，优先复用当前图纸已有且能正常显示中文的文字样式。
2. 需要新建样式时，先检查本地字体。Windows 可用 `Get-ChildItem $env:WINDIR\Fonts` 查看字体文件。
3. 常用优先级：`宋体/SimSun` 用于一般中文说明，`黑体/SimHei` 用于标题，`仿宋/FangSong` 或 `仿宋_GB2312` 用于标题栏和正式说明，`Microsoft YaHei` 仅在项目允许时用于屏显友好的图纸。
4. 如果使用 SHX 字体，必须确认图纸环境存在对应 `.shx` 文件。不要引用不存在的 `HZTXT.SHX`、`TSSDENG.SHX`、`ROMANS.SHX` 等字体。
5. 创建文字样式时使用 `create_text_style(name, font, height, width)`，然后用 `set_current_text_style`。中西文混排要检查宽度比例和输出 PDF 是否乱码。

## 文字高度

按打印高度控制文字。若没有注释比例/布局文字机制，在模型空间中文字高度约等于 `打印高度 x 出图比例分母`。

| 用途 | 打印高度建议 |
|---|---:|
| 尺寸文字、普通注释 | 2.5 或 3.5 |
| 房间名、设备编号 | 3.5 或 5.0 |
| 图名、详图名 | 5.0 或 7.0 |
| 图纸主标题 | 7.0 或 10.0 |

同一套图纸内保持文字层级一致。不要为了“看得见”随意缩放单个文字实体。

## 标注样式

- 尺寸线、尺寸界线、箭头/斜短线、文字高度、精度、单位格式应统一。
- 建筑平面尺寸通常按外部总尺寸、轴线尺寸、洞口/构件尺寸分层组织。
- 结构和机电专业必须保证标注值、构件编号、管径/截面/标高/回路号与几何和表格一致。
- 覆盖尺寸文字前，先用 `get_dimension_measurement` 或测量工具确认真实值。
